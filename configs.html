Configuration Commands Set up Maven's settings in your server in your server through Maven's commands.
!config color [#000000] Add your primary embed color
!config color2 [#000000] Add your secondary embed color
!config soloc [#channel] Add your solo scrim announcement channel
!config duoc [#channel] Add your duo scrim announcement channel
!config trioc [#channel] Add your trio scrim announcement channel
!config squadc [#channel] Add your squad scrim announcement channel
!config region [region] Add your region for the scrims you want to host
!config host [@role] Add your host role (role that can only use Scrims commands)
!config amount [amount/prize] Add the amount of money, or prize, the winner of the scrim wins
!config rules [rules] Add the scrim rules
!config image [image-link] Add the thumbnail for the embed in your scrim announcement
!config mode [Arena/Regular] Add the game mode for the scrims you want to host
